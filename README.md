
# Résumé

This repository contains my resume, written in LaTeX. It is designed to be easy to read and understand, highlighting my skills and experience in an organized and professional manner.

## How to Use

If you find this template helpful, feel free to fork it and use it as a starting point for your own resume. The template is open-source and available under the MIT license, which means you are free to use, modify and distribute the template as you see fit.

To use the template, you will need to have LaTeX installed on your system. You can download it from the official website<br>
[https://www.latex-project.org/get](https://www.latex-project.org/get/)

If you want to make any changes, you can edit the `résumé.tex` file and then use a LaTeX compiler to generate a pdf.

If you have any questions or need help getting started, please don't hesitate to reach out to me.

Thank you for checking out my repository, and I hope you find it useful!
